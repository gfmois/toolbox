Toolbox CLI · Modular platform

The internal CLI platform that scales with your team.

Toolbox lets you ship internal CLI products as plugins inside a single binary. Less friction, higher adoption, and centralized updates. Built for teams shipping lots of tooling with a polished developer experience.

Open repository Install guide
$ toolbox list ✔ auth ✔ deploy ✔ observability ✔ scripts $ toolbox deploy release --env prod

Install in minutes

1. Install the core

toolbox --version

2. Add plugins

toolbox plugins install auth toolbox plugins install deploy

3. List commands

toolbox list

Updates without pain

Update the core and every plugin from a single command. No manual scripts, no hidden dependencies.

toolbox self update toolbox plugins update --all

How it works

Stable core

Responsible for plugin loading, command registration, and safe execution.

Versioned plugins

Each team ships features on their own cadence without affecting others.

Centralized delivery

Visibility, governance, and auditability across every internal tool.
