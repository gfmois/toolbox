Toolbox CLI · Plataforma modular

La CLI interna que escala con tu equipo.

Toolbox te permite distribuir productos internos como plugins dentro de un único binario: menos fricción, más adopción y actualizaciones centralizadas. Diseñado para equipos que construyen muchas herramientas y quieren una experiencia impecable.

Instalación en minutos

1. Instala el core

toolbox --version

2. Agrega plugins

toolbox plugins install auth toolbox plugins install deploy

3. Lista comandos

toolbox list

Actualizaciones sin dolor

Actualiza el core y los plugins desde un único comando, sin scripts manuales ni dependencias ocultas.

toolbox self update toolbox plugins update --all

Cómo funciona

Core estable

Responsable de la carga de plugins, registro de comandos y ejecución segura.

Plugins versionados

Cada equipo versiona su plugin y publica funcionalidades en ciclos propios.

Distribución centralizada

Auditoría, control y visibilidad total sobre lo que usa el equipo.
